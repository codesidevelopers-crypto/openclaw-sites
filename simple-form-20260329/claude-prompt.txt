Собери очень простой публичный сайт на Gatsby + TypeScript + styled-components + typed theme.

Задача:
- одна страница
- фактически один экран / один основной блок по центру
- короткий заголовок на русском
- краткий подзаголовок
- простая форма заявки
- поля формы: Имя, Телефон, Комментарий
- submit должен идти на /api/submit
- payload должен быть плоским, с русскими human-readable ключами
- добавь скрытое поле/значение или явный ключ для таба/источника, если это уже принято в проекте, но не усложняй
- после успешной отправки покажи простое сообщение об успехе
- дизайн намеренно минималистичный, аккуратный, современный
- без лишних секций, без сложной анимации, без тяжелого декора

Технические требования:
- TypeScript без any
- Gatsby
- styled-components
- typed theme
- структура должна быть достаточно полной, чтобы проект собирался и работал
- если нужно, создай необходимые файлы src/pages/index.tsx, стили, theme, gatsby-config и т.д.
- не добавляй новые нестандартные зависимости без необходимости

Контент:
- заголовок: "Оставьте заявку"
- подзаголовок: "Свяжемся с вами и обсудим задачу"
- кнопка: "Отправить"

Сделай результат готовым к локальной сборке.
